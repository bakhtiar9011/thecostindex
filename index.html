Privacy Policy

Effective Date: 08/07/2025

This Chrome Extension ("the Extension") respects your privacy. This policy explains what information we collect and how it is used.

Information Collected

  • Title and Input Text: The extension uses the video title to generate a YouTube description and fetch tag suggestions. No titles or user data are stored.
  • Google Suggest: For tag generation, we use publicly available autocomplete data from suggestqueries.google.com.
  • OpenAI API: The title is sent to a backend that interacts with OpenAI's API to generate descriptions. No personal data is stored or logged.

Data Storage

We do not store, collect, or sell any user data. All processing is done in real-time and transiently.

Permissions

The extension requests access only to the YouTube Studio upload page and two third-party APIs to enhance your workflow:

  • https://studio.youtube.com/ – To display buttons on the YouTube upload/edit interface.
  • https://suggestqueries.google.com/ – To fetch real-time tag suggestions.
  • https://api.openai.com/ – To generate YouTube descriptions based on the video title.

Third-Party Services

This extension uses external services (OpenAI and Google Suggest) only to enhance content generation and does not share any user data with third parties.
